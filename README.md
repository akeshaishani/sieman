# Sieman

A modern, responsive corporate website built to showcase a company's services, projects, expertise, and brand identity.

## ✨ Overview

Sieman is a high-performance corporate website designed with a focus on modern UI/UX, accessibility, SEO, and responsive design.

The project demonstrates best practices for building scalable marketing websites using the latest frontend technologies.

## 🚀 Features

- Modern responsive design
- Mobile-first development
- Server-side rendering
- SEO optimized
- Reusable UI components
- Smooth animations
- Fast page loading
- Contact section
- Services showcase
- Projects portfolio
- Company information pages
- Optimized assets
- Clean folder structure

## ⚡ Getting Started

### Clone the repository

```bash
git clone https://github.com/akeshaishani/sieman.git
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

## 📦 Build

```bash
npm run build
```

## 🚀 Production

```bash
npm run start
```
